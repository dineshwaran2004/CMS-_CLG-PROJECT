# Event Management and Hall Booking System

This project is a **web-based Event Management and Hall Booking System** developed using **HTML, CSS, PHP, and MySQL**.
The system allows users to book event halls, select food menus, and manage different types of functions such as weddings, birthdays, and other celebrations.

This project demonstrates basic **full-stack web development skills including frontend design, backend development, and database management.**

---

## Project Features

* User login system
* Event hall booking
* Wedding booking
* Birthday function booking
* Puberty / Valagaappu function booking
* Food menu selection
* Worker management
* Admin dashboard
* Customer feedback system
* Event location viewing

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/dineshwaran2004/event-management-system.git
```

---

### 2. Move Project to XAMPP

Copy the project folder into:

```
xampp/htdocs/
```

---

### 3. Import Database

1. Open **phpMyAdmin**
2. Create a new database
3. Import the file:

```
127_0_0_1 (2).sql
```

---

### 4. Run the Project

Open your browser and go to:

```
http://localhost/event-management-system
```
